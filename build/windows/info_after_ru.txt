========================================
VeilBox — что дальше?
========================================

Установка завершена. Ниже — краткая памятка по первым шагам, настройке и полезным ссылкам.

1) Первый запуск
— Запустите «VeilBox» из меню «Пуск» или с ярлыка на рабочем столе.
— При первом запуске вас встретит окно добавления профиля/подключения.
— Если при установке вы выбирали «Импорт профиля», он уже доступен в списке.

2) Где лежат файлы
— Программа: {app}\VeilBox.exe
— Служебные файлы: {app}\core
— Пользовательские данные (профили, кэш, логи): %LOCALAPPDATA%\VeilBox

3) Профили/конфиги
— Добавление: в приложении откройте «Настройки → Профили → Добавить», вставьте параметры (VLESS/Reality/…).
— Импорт из файла JSON: «Настройки → Импорт», выберите файл, или поместите его в %LOCALAPPDATA%\VeilBox\profiles\ и перезапустите приложение.
— Редактирование/экспорт: «Настройки → Профили → … → Экспорт».

4) Автозапуск и трей
— Включить автозапуск: «Настройки → Общие → Запускать при входе в систему».
— Запуск свернутым: «Настройки → Общие → Стартовать в трее».
— Значок в трее: быстрые действия — подключиться/отключиться, выбрать профиль, открыть окно.

5) Сеть и брандмауэр
— Если вы не разрешали правила брандмауэра при установке, Windows может спросить при первом подключении — разрешите для частных сетей.
— Для ручной настройки: «Параметры Windows → Брандмауэр → Разрешения для приложений → VeilBox».

6) Обновления
— Рекомендуется периодически устанавливать обновления (через меню «Справка → Проверить обновления» или переустановкой поверх).
— Обновление поверх не удаляет ваши профили и настройки.

7) Диагностика и логи
— Логи находятся в %LOCALAPPDATA%\VeilBox\logs\
— Для ускоренной помощи приложите последние файлы логов и укажите версию VeilBox (см. «Справка → О программе»).

8) Удаление
— «Параметры Windows → Приложения → Приложения и возможности → VeilBox → Удалить».
— В деинсталляторе можно выбрать: удалить ли пользовательские данные из %LOCALAPPDATA%\VeilBox.
  (По умолчанию данные сохраняются — это удобно при повторной установке.)

9) Требования и совместимость
— Windows 10/11 x64. Для UI используется WebView2 Runtime (при отсутствии устанавливается автоматически).
— Для лучшей стабильности обновляйте систему и сетевые драйверы.

10) Полезные советы
— Если подключение медленное/нестабильное: попробуйте другой профиль/сервер, протокол (например, Reality gRPC/UDP), или отключите лишние сетевые фильтры/антивирусные функции.
— При ошибках сертификата/рукопожатия проверьте системное время и параметры Reality (SNI/ServerName, ключи).
— Для корпоративных сетей может потребоваться настройка прокси/исключений.
