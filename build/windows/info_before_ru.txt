========================================
VeilBox — информация перед установкой
========================================

Спасибо, что выбираете VeilBox. Ниже — кратко о том, что будет установлено, какие права нужны и где будут храниться данные.

1) Что установится
— Программа: VeilBox (VeilBox.exe)
— Служебные файлы: папка {app}\core
— Ярлыки: меню «Пуск» и (по выбору) рабочий стол
— (Опционально) Импорт профиля/конфига, если вы включите соответствующую задачу

2) Требования
— ОС: Windows 10/11 x64
— Права: администратор (для записи в Program Files и добавления правил брандмауэра — по вашему выбору)
— WebView2 Runtime: при отсутствии система предложит установить (небольшая загрузка из интернета)

3) Что будет изменено в системе (по вашему выбору)
— Ярлык в меню «Пуск» и (опционально) на рабочем столе
— (Опционально) Автозапуск VeilBox при входе в систему (HKCU\Software\Microsoft\Windows\CurrentVersion\Run)
— (Опционально) Правила брандмауэра Windows для VeilBox.exe (входящие/исходящие «Разрешить»)

4) Хранение данных
— Приложение: {app}
— Пользовательские данные (профили, кэш, журналы): %LOCALAPPDATA%\VeilBox
  * Эти файлы остаются при удалении программы, если вы явно не согласитесь удалить их в деинсталляторе.

5) Сеть и конфиденциальность
— VeilBox устанавливает соединения только с указанными вами серверами согласно выбранным профилям.
— Технические журналы хранятся локально и могут включать диагностические записи (без персональных данных, если вы их не укажете).
— Дополнительно смотрите «Лицензионное соглашение» и «Политику конфиденциальности» (при наличии).

6) Обновления
— Программа может предлагать обновления. Установка возможна поверх (AppId一致 обеспечивает корректный апгрейд без дубликатов).

7) Удаление
— Удалить VeilBox можно через «Приложения и возможности» Windows.
— В процессе деинсталляции вам предложат (опционально) удалить пользовательские данные из %LOCALAPPDATA%\VeilBox.

8) Известные ограничения
— Цветовую схему установщика определяет Windows (светлая/тёмная тема). Графические элементы VeilBox подстраиваются автоматически.
— Для стабильной работы рекомендуется актуальная версия Windows и сетевые драйверы.

Нажимая «Далее», вы подтверждаете, что ознакомились с информацией выше и перейдёте к просмотру Лицензионного соглашения (EULA). Установка продолжится только после принятия условий лицензии.
